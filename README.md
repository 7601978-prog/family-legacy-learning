# Family Legacy Learning

Одностраничник закрытой образовательной программы **Family Legacy Learning** — программы Tumar Family Office под руководством Галии Нурбековой.

## Стек

Чистый HTML + CSS + минимум JS. Без сборки, без зависимостей. Готов для GitHub Pages.

## Структура

- `index.html` — разметка одностраничника с `data-i18n` атрибутами
- `styles.css` — стили (quiet luxury: navy + gold, serif заголовки, sans-serif тело) + RTL
- `i18n.js` — словари локализации RU / KZ / AR / EN, переключатель языков
- Шрифты — Cormorant Garamond и Inter (Google Fonts), IBM Plex Sans Arabic для арабского

## Локализация

Сайт работает на четырёх языках: **русский**, **қазақша**, **العربية** (RTL), **English**.
Переключение — кнопками в шапке. Выбор сохраняется в `localStorage`,
при первом заходе определяется по `navigator.language`.

> ⚠️ **RU и EN** — проверенные переводы.
> **KZ и AR** — черновые. По требованиям PRD (§3.2) тексты обязаны
> писать носители: для арабского — с учётом форм вежливости региона Залива.
> Перед продом — обязательное ревью носителей.

## Локальный запуск

Открыть `index.html` в браузере. Или через простой сервер:

```bash
python3 -m http.server 8000
# затем http://localhost:8000
```

## Деплой

Опубликован через GitHub Pages из ветки `main`, корень репозитория.
